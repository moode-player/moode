AirPlay

Settings
Use SoX for best quality or Basic which consumes fewer CPU resources.
Bit depth output by AirPlay receiver. Default is 16 bit.
Sample rate output by AirPlay receiver. Default is 44.1 kHz. Conversion to rates greater than 44.1 kHz is done using simple upsampling and therefore these higher rates should only be used if the audio device doesn't support 44.1 kHz rate.
Allow session to be interrupted by another AirPlay sender. Default is No.
Wait for this number of seconds after a source disappears before terminating the session and becoming available again. Default is 120.
Set this offset to compensate for a fixed delay in the audio back end e.g. if the output device delays by 100 ms, set this to -0.1.
If set too small, buffer underflow occurs on low-powered machines. If set too long the response times with software mixer become annoying. Default is 0.15.
