Settings
Default is SoX for best quality, alternative is Basic which consumes fewer CPU resources.
Bit depth output by AirPlay receiver. Default is 16 bit.
Sample rate output by AirPlay receiver. Default is 44.1 kHz.
Conversion is done using simple upsampling and should only be used if the audio device doesn't support 44,100 rate.
Allow session to be interrupted by another AirPlay sender. Default is No.
Wait for this number of seconds after a source disappears before terminating the session and becoming available again. Default is 120.
Set this offset to compensate for a fixed delay in the audio back end. E.g. if the output device delays by 100 ms, set this to -0.1.
If set too small, buffer underflow occurs on low-powered machines. Too long and the response times with software mixer become annoying. Default is 0.15.
