Settings
The Automatic setting defaults to using Triangular for formats S16, S24, S24_3 and None for all other formats.
Range of the volume control in dB from 0-100. Default is 60. This setting is used to adjust the steepness of the volume curve and influences the usable range of the volume slider.
Initial volume in % from 0-100. Set to -1 to use the cached volume or if none exists then default to 50.
Adjusts the volume of all songs played so that they sound as though they are of equal loudness.
Volume boost in dB from -10 to 10. Default is 0 dB.
Threshold in dBFS to prevent clipping. Default is -2.0 dBFS.
Attack time in milliseconds in which the dynamic limiter is reducing gain. Default is 5 ms.
Release or decay time in milliseconds in which the dynamic limiter is restoring gain. Default is 100 ms.
Knee steepness of the dynamic limiter. Default is 1.0.
Autoplay similar songs when the playlist ends.
