Select which output device will receive audio from the selected source.
Select which input source provides audio and which output device will receive it. The default input source is Local (MPD) and the default output device is the locally attached device (DAC, Amp or Transport).

If the audio device offers input sources for example Analog or S/PDIF they will be listed. These input sources use Hardware volume so be sure to set MPD Volume control to Hardware or Disabled (0dB). If MPD Volume control is set to Hardware moOde can function as a preamp.
