Backup options

Specify which parts of the system will be included in the backup zip file.
A script file may be added that performs custom actions during the restore process. It is run during moOde startup as part of restoring the Configuration settings.

Restore options

Specify which parts of the system will be restored.
The Yes/No option buttons are set based on the contents of the selected backup zip file. If a script file was added to the backup it will be run during moOde startup as part of restoring the Configuration settings.


$_togglebtn_restore_system Restore configuration settings and preferences including a backdrop image if any.
$_togglebtn_restore_camilladsp Restore CamillaDSP pipeline configurations and IR files.


$_togglebtn_restore_radiostations_moode Restore moOde radio stations.
$_togglebtn_restore_radiostations_other Restore user added radio stations.
Select - Select and upload a backup file that will be used for the restore process.
Reset - Clear uploaded files from the temp directory and reset the option buttons to No.
$_imported_backupfile


If configuration settings are part of the restore an automatic reboot will be performed during moOde startup.
