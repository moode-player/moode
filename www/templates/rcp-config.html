Radio Cover+ Config

Radio Cover+ provides advanced search and analysis that greatly increases the probability of finding the correct cover art for the currently playing track. The implementation thats included is a Command Line Interface (CLI) port of the full moOde-radio-plus (MR+) daemon. It can be selected in the Cover Art section of Preferences. Visit the moOde-radio-plus (MR+) project for more information.
