Music Library

Remote NAS Sources

Create a remote NAS source when the song files reside on another computer in the network. After saving or removing a NAS source select "Update library" from the Menu when in Playback or Library views.

Remote NAS source

Remote NAS sources Unmount all remote NAS sources then re-mount them.
$_select[fs_mountmon_on] $_select[fs_mountmon_off]
Mount monitor Attempt to automatically restore Remote NAS Source mounts that become temporarily unavailable due to network or file sharing service interruption. Due to the protocols used in SMB and NFS there can be long timeouts when checking a mount or performing a re-mount. Refer to the file /var/log/moode_mountmon.log when Debug logging is on.
Local NVMe Drives

Local NVMe drives must be manually mounted or un-mounted. Click MOUNT to mount a new NVMe drive. After mounting or un-mounting the drive select "Update library" from the Menu when in Playback or Library views. Note that only EXT4 formatted drives are supported.

Local NVMe drive
$_nvme_mounts
Local NVMe drive A List of NVMe drives will be presented showing the device ID and in parenthesis its volume label or a status for example "Unformatted" or "No label".
Local SATA Drives

Local PCIe attached SATA drives must be manually mounted or un-mounted. Click MOUNT to mount a new SATA drive. After mounting or un-mounting the drive select "Update library" from the Menu when in Playback or Library views. Note that SATA drives must be formatted outside of moOde prior to mounting.

Local SATA drive
$_sata_mounts
Local USB Drives

Local USB drives are auto-mounted when they are inserted and auto-unmounted when they are removed. After inserting or removing the drive select "Update library" from the Menu when in Playback or Library views.

Music Database

The music database consists of the MPD database, Library tag cache and Album cover thumbnail cache. The Library tag cache contains track metadata extracted by MPD plus custom metadata generated by moOde.

Music database Regenerate the MPD database, Library tag cache and album cover thumbnail cache.
Library tag cache The Library tag cache contains all music metadata and will be created when returning to Playback or Library view.
$_select[cuefiles_ignore_on] $_select[cuefiles_ignore_off]
Ignore CUE files This can help prevent duplicate tracks entries when a .cue file is present in the album directory. After changing this setting return to Playback or Library then select "Update library" from the Menu.
Thumbnail Generator

Album cover thumbnail images are used to populate the Library views, Playbar and the Queue. The Cover Art section of Preferences contains options for these thumbnail images.

Album cover thumbnail cache Regenerate the album cover thumbnail cache by first deleting it and then re-populating it with newly generated thumbnails.
