General
Check that the input format is supported. Set this option to No if a control point or media server sends good audio data with bad metadata.
TIDAL
Your Tidal login name.
The password for your Tidal account. NOTE: Single and double quotes are not allowed.
Stream quality: 'low' and 'high' are AAC streams, 'lossless' is FLAC and will only work if your subscription allows it.
Qobuz
Your Qobuz login name.
The password for your Qobuz account. NOTE: Single and double quotes are not allowed.
Stream quality: 320K (MP3) or lossless (FLAC) if your subscription allows it.
Google Play

The Google Music feature is enabled but the API must be manually installed.
Run the command "sudo pip install gmusicapi" from an ssh session.

Google Play
Your Google Music login name (Gmail address).
The password for your Google Music account. NOTE: Single and double quotes are not allowed.
Stream quality: 320K, 160K or 128K (MP3).
Music device id. Your system MAC address will be used by default, but this will count against the maximum count of registered devices for this account. You can optionally set this value to the device ID from a phone or tablet on which you also use Google Play Music.
