Options
Audio routing Route audio to the Local audio device or a remote Bluetooth speaker.
PCM buffer Lower values reduce latency but may cause dropouts.
SBC quality Higher settings improve quality but may cause dropouts.
Low: Mono 114 kbps, Stereo 213 kbps
Medium: Mono 132 kbps, Stereo 237 kbps
High: Mono 198 kbps, Stereo 345 kbps
XQ: SBC Dual Channel HD 452 kbps
XQ+: SBC Dual Channel HD 551 kbps>
ALSA output mode Standard: Use the Output mode setting from Audio Config. DSP and Peppy are supported.
Compatibility: Does not support DSP or Peppy but may help if there is no sound output.
Controller mode Sets the controller to the specified transport. The default mode is "Dual" which enables both BR/EDR and LE transports.
