Server
Most NAS devices support SMB (Samba). NFS is only for UNIX/Linux. UPnP is for UPnP/DLNA media servers.
NOTE: Unsuccessful NFS mounts can cause up to a 1-2 min timeout. This screen will appear to hang when this happens.
NAS devices: the format is Host/Share or IP_Addr/Share.
UPnP devices: the format is Host/Folder.
SCAN generates a list of SMB (Samba) shares or UPnP folders.
For UPnP folders, select a folder then SCAN to display its sub-folders.
