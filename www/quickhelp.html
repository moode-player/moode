Navigation

The UI has two main parts: Library and Configuration. The Library has four views for selecting content including Radio, Folder, Tag and Album and a Playbar for controlling playback. Toggling between the Playbar and full screen Playback is accomplished by tapping on the cover or the bar icon. The Configuration screens are accessed via the Main Menu.

Special Buttons

Most buttons are self explanatory but several perform unique functions.

  • CoverView Display full screen CoverView screen saver. Tapping the cover in CoverView displays play controls. Tapping anywhere else dismisses CoverView.
  • Random Play random tracks from the Playlist (default) or the entire Music Library if Auto-shuffle is ON (see Audio Config).
  • Consume Remove the track from the Playlist after it finishes playing. Consume mode is automatically enabled when Auto-shuffle is ON.
  • Repeat Repeat the Playlist or if Single is enabled then repeat the track.
  • Single Used in conjunction with Repeat to repeat a single track.
  • Last track Toggle between the currently playing track and the previously played track.
  • Random album Instantly play a randomly selected album when in full screen Playback or if in the Library Tag or Album view select a random Album for review.
  • Favorite Add the currently selected track in the Playlist to the Favorites playlist. Playlists are listed in the Library Folder view.

Active Elements

There are several non-button elements in the UI that perform an action when tapped.

  • Instant Play Tapping on the cover in Library Radio or Album views instantly plays the item by performing a Clear/Play.
  • Renderer Active screens Tapping on the title in the center of Bluetooth, Analog Input or S/PDIF Input Active will launch their respective Config screens which allow the input/output settings to be changed.
  • Time knob Tapping on the time while a song file is playing will toggle between counting the time up or down. Tapping or dragging the time ring moves forward or backward in the song.
  • Volume knob Tapping on the numeric volume level toggles the mute state. Tapping ahead on the volume ring advances the volume by 10. Tapping the (- +) buttons adjusts volume by 1 step.
  • Song title Tapping the song title in full screen Playback launches a Google search in a separate Browser tab. On Smartphones it scrolls the Playlist to that item.
  • Swipe gestures On Smartphones swiping up scrolls the Playlist into view and tapping on the Playbar cover returns to Playback. In the Library tag view swiping sideways shows/hides the Genres column.
